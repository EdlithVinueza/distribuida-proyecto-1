Remover todos los contenedores, redes y volúmenes

docker compose -f "/home/edlith/Documentos/UCE 26-26/DISTRIBUIDA/distribuida-proyecto-1/deployment/telemetria/docker-compose.yml" -p telemetria down

Levantar todos los contenedores, redes y volúmenes

docker compose -f "/home/edlith/Documentos/UCE 26-26/DISTRIBUIDA/distribuida-proyecto-1/deployment/telemetria/docker-compose.yml" -p telemetria up -d

Detener servicio local de PostgreSQL (para liberar el puerto 5432)
sudo systemctl stop postgresql

Evitar que PostgreSQL local inicie automáticamente al encender el equipo
sudo systemctl disable postgresql

Habilitar nuevamente PostgreSQL local (si se necesita a futuro)
sudo systemctl enable postgresql

Comandos Docker Compose para gestionar servicios individuales (ej: authors, books, customers, db-server)
Detener un servicio específico:
docker compose -f "/home/edlith/Documentos/UCE 26-26/DISTRIBUIDA/distribuida-proyecto-1/deployment/telemetria/docker-compose.yml" -p telemetria stop [nombre-del-servicio]

Iniciar un servicio específico que fue detenido:
docker compose -f "/home/edlith/Documentos/UCE 26-26/DISTRIBUIDA/distribuida-proyecto-1/deployment/telemetria/docker-compose.yml" -p telemetria start [nombre-del-servicio]
